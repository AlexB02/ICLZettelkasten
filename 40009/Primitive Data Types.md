A [[Primitive Data Types|PDT]] is a [[Data Type]] which has no methods & purely stores a value.

[[Java]] has 8 primitive data types.
- `boolean`
- `char, byte, short, int, long`
- `float, double`

Each primitive type has:
- a set of values
- a data representation
- a set of operations
